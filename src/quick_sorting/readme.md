# 复杂度分析
## 重新构造数组方式的 quick_sorting
### 时间复杂度
最好情况与平均情况: O(NlogN) 
分析：
logN(层数) * N(每层)
最差情况: O(n2)

最差情况：N2（分块时，每次都有一边是 0 个，导致层数为 N）

### 空间复杂度
最好情况与平均 O(NlogN)
logN(层数) * N(每层)

最差情况: O(N2)（层数为 N）


## 不重新构造数组方式的 quick_sorting
### 时间复杂度
最好情况与平均情况: O(NlogN) 
分析：
logN(层数) * N(每层)
最差情况: O(n2)

最差情况：N2（分块时，每次都有一边是 0 个，导致层数为 N）

### 空间复杂度
最好情况与平均 O(logN)
(3 + 6 + ... + N) = (2 + N) * logN/2 = N * logN

最差情况: O(n2)（层数为 N）
(3 + 6 + ... + N) = (2 + N) * N/2 = N2
